# PROJECT: Signaling server for webrtc use WEBSOCKET SECURE (WSS) WITH JWT
=======================================================

## Prerequisites
    * Nodejs version = 10.4.1 (important)
    * NPM version = 6.3.0 (important)

## Installing
    * npm install

## Use
    * Implement as tutorial (1)
    * node index.js
    * Access: wss://domain:port, or https://domain:port
